DATA FLOW:
═══════════════════════════════════════════════════════════════════════════════════

  1️⃣  User Login         : MSP Engineer → Entra ID (MSAL Auth)
  2️⃣  Access Token       : Entra ID → App Service (Frontend)
  3️⃣  API Requests       : Frontend → Functions App (Backend)
  4️⃣  Backend Processing :
      ├─ 📧 Fetch Mails  : Functions → MS Graph API (Outlook)
      ├─ 🤖 AI Generation: Functions → GitHub Models (GPT-4o)
      ├─ 🔑 Read Secrets : Functions → Key Vault (Managed Identity)
      ├─ 💾 Store RCA    : Functions → Storage Account (Blob)
      └─ 📊 Send Logs    : Functions → Application Insights
  5️⃣  Share Results     : Functions → Slack Webhook
